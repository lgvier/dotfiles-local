# dotfiles-local

## Step 1:
Clone this repo
`git clone https://github.com/lgvier/dotfiles-local.git ~/dotfiles-local`

## Step 2:
Install https://github.com/thoughtbot/dotfiles

## Step 3:
Install TPM:
```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

All set!
To update local configs: `rcup`
